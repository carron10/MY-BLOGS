How to build a website

A website is an application that runs through a web browser. This is kind of an application/system that anyone connected to the internet with an cellphone,computer or any device with a browser can access it.

Lets see some examples of websites:
  1. University or College website.
    Your college or University uses a website to update students on different information e.g student results, news or etc.
  2. Social media website.
    This is are kindly of websites that lets people communicate and share information e.g Facebook.
  3. Search engine website.
    This is a website that people can use to search for information, they don't real provide information but a summary and a source of what you are looking for. E.g Google search engine.
  4. Portiforlio website.
    This are website that people built to show their skills,projects or any information about them. For example I can build a one page website that acts as my CV/Resume.
  5. Ecommerce Website.
    Companies uses website to sell their products or services online.
  6. Dashboard.
    You can use a website to manage your files or kind of system information.

In these series of articles I will show you how to built a website from scratch,resources that are needed and how to host your website on internet so that everyone can access it. Through your learning process you will come across to terms a Server and Client or Server and User.

A server is a computer were your website is hosted(stored or served) and a client is a device that you are using to access a website, in this case your cellphone/computer. A user is a person who is accessing that website, in this case you are a user. So, to build a website first of all you need a computer to host a website, this can be your Laptop or your cellphone or other computers designed to be servers. You can use your computer as both for server and client, as long as there is chrome or any other browser installed on your device.

There are two types of websites Static and Dynamic website. A static website is a website which does not perfume any computation, this website just that contains only static files that can only be executed on the client side.

In this article will start by building a static website then a Dynamic website. There are many programming Languages that we can use to build up a website, these are but not limited to

  • Java
  • python
  • php
  • javascript
In this article we will use PHP,javascript,html and css to build a Portiforlio website.
Setting up environment
Installing PHP
To get started you need to install php on your machine. PHP is a programming language that runs on the server side, it is the one which feed the client side with what they want. e.g If you visit a website like google.com, there is a software on their server which is going to feed you with a webpage you are looking for.
Installing Vscode.
Visual studio code is an IDE(intergrated development environment) that we use to open how files and edit the codes. It works like Codeblocks if you are familiar with it.
Installing a browser.
if you to have a web browser on your PC you can download and install chrome.

Now we are done with setting up environment, you can now read the next article which will teach you more.
